
## API Reference

#### Register a new User 

```http
  POST /api/items
```

| Parameter | Type     | Description                |
| :-------- | :------- | :------------------------- |
| `Usuario` | `string` | **Required**. User entity to save |

#### Get item

```http
  GET /usuarios/{id}
```

| Parameter | Type     | Description                       |
| :-------- | :------- | :-------------------------------- |
| `id`      | `string` | **Required**. Id of item to fetch |
